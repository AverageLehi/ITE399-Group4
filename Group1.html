There are many formatting tags in HTML. These tags used to make text bold, italicized, or underlined.
There are almost 14 options available about how text appears in HTML.

In HTML, the formatting tags are divided into two categries:

Physical tags: These tags used to provide the visual appearance to the text.
Logical tag: These tags used to add some logical or semantic value to the text.
