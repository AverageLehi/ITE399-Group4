Embed & NoEmbed Tags

The <embed> tag includes external resources with a <noembed> fallback:

Embed PDF

<p>Your browser does not support PDF embedding. <a href="sample.pdf">Download the PDF file</a> instead.</p>

Embed Image

<p>Your browser does not support embedded images. Check your browser settings.</p>

Embed Video

<p>Your browser does not support embedded videos. <a href="sample.mp4">Download the video</a> instead.</p>

Frameset & Frame Tags

⚠️ Note: <frameset> and <frame> tags are deprecated in HTML5. Modern websites use <iframe> instead.

Traditional Frameset Structure (Deprecated)

<frameset rows="50%,50%">
    <frame src="top.html" name="top">
    <frameset cols="30%,70%">
        <frame src="left.html" name="left">
        <frame src="right.html" name="right">
    </frameset>
</frameset>

Modern Alternative Using iframe

Complete Tag Summary

  • <marquee> - Creates scrolling text (with direction, speed, behavior attributes)
  • <ul> - Unordered list (bullet points)
  • <ol> - Ordered list (numbered)
  • <dl> - Description list (term and definition pairs)
  • <table> - Table container
    • <thead> - Table header section
    • <tbody> - Table body section
    • <tfoot> - Table footer section
    • <caption> - Table caption
    • <tr> - Table row
    • <th> - Table header cell
    • <td> - Table data cell
  • <embed> - Embeds external resources (PDF, images, video)
  • <noembed> - Fallback content if embed is not supported
  • <frameset> - Creates frame layout (deprecated in HTML5)
  • <frame> - Individual frame (deprecated in HTML5)
  • <iframe> - Modern alternative for embedding external content
