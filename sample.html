6. Embed & NoEmbed Tags

Embed tag is used to include external resources:

Embed PDF (if supported):

<p>Your browser does not support PDF embedding. <a href="sample.pdf">Download the PDF file</a> instead.</p>

Embed Image:

<p>Your browser does not support embedded images.</p>

Embed Video (if supported):

<p>Your browser does not support embedded videos. <a href="sample.mp4">Download the video</a> instead.</p>

7. Frameset & Frame Tags (Deprecated)

Note: Frameset and frame tags are deprecated in HTML5. Modern websites use iframe instead.

Traditional frameset structure would look like:

<frameset rows="50%,50%">
    <frame src="top.html" name="top">
    <frameset cols="30%,70%">
        <frame src="left.html" name="left">
        <frame src="right.html" name="right">
    </frameset>
</frameset>

Summary of All Tags Used:

  • <marquee> - Creates scrolling text (with direction, speed, behavior attributes)
  • <ul> - Unordered list (bullet points)
  • <ol> - Ordered list (numbered)
  • <dl> - Description list (term and definition pairs)
  • <table> - Table container
    • <thead> - Table header
    • <tbody> - Table body
    • <tfoot> - Table footer
    • <tr> - Table row
    • <th> - Table header cell
    • <td> - Table data cell
  • <embed> - Embeds external resources (PDF, images, video)
  • <noembed> - Fallback content if embed is not supported
  • <frameset> - Creates frame layout (deprecated)
  • <frame> - Individual frame (deprecated)
  • <iframe> - Modern alternative to frames
